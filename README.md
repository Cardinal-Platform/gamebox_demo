本项目为 Cardinal 平台 AWD 题目示例环境。其中包含一个 Apache + PHP 搭建的 Web 站点，以及一个 MySQL 数据库。

该仓库仅用作 Cardinal 靶机环境参考。仓库本身**不是**一道 AWD 赛题。
